## animation
3つのアニメーションがある.
| keyword | description |
| --- | --- |
| layers | ボードを継承せず, それが新しいベースになる |
| scenarios | ベースlayerを継承するボード |
| steps | 前のstepを継承するボード |

これまでのダイアグラムを書いていた場所をroot boardという.
また, ボードが一つなので, single boardだった.

D2は複数のボードを持つことができる.
```plaintext
x -> y # これは root board
layers: {
  # root boardから何も継承されていない"numbers"という新しいboardができた.
  numbers: {
    1 -> 2
  }
}
```

```d2 animateInterval=1000
x -> y
layers: {
  numbers: {
    1 -> 2
  }
}
```
ここでは, root boardとnumbers boardがある. これらは同時には表示されない.

### layer
ボードを継承せず, それが新しいベースになる
```d2 animateInterval=1000
style.fill: transparent
How does the cat go? 0

layers: {
  cat: {
    style.fill: transparent
    Meooooooooowwww 1
  }
  cat: {
    style.fill: transparent
    Meooooooooowwww 2
  }
  dog: {
    style.fill: transparent
    Meooooooooowwww 3
  }
}
```

```plaintext
  }
```

```d2 animateInterval=1000

direction: right
x -> y

  A :{
```

```plaintext
      animated: true
    }
  }
}
```

### steps
前のstepを継承するボード

```d2 animateInterval=1000
How does the cat go? 0

  1: {
    style.fill: transparent
    Meooooooooowwww 1
  }
  2: {
    style.fill: transparent
    Meooooooooowwww 2
  }
  3: {
    style.fill: transparent
    Meooooooooowwww 3
  }
}
```
